# Casuta dintre brazi — Meniu QR

Site static cu meniul terasei **Casuta dintre brazi**.

## Cum îl deschizi

Deschide fișierul `index.html` în browser (dublu-click) sau rulează un server local:

```bash
npx serve .
```

## Structură

- `index.html` — pagina meniului
- `style.css` — stiluri (temă alb + roșu)
- `script.js` — filtrare pe categorii
- `LOGO CDB.jpg` — logo terasă

## Categorii

- **Mâncare** — preparate calde, preparate la grătar, chiflă/pâine
- **Dulciuri** — kürtős kalács, kürtős cu înghețată, înghețată
- **Băuturi** — calde, sucuri, apă, bere
